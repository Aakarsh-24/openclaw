# Quick Reference: New CLI Tools

## ripgrep (`rg`)
Ultra-fast search across files.
```bash
rg "pattern"                    # Search current dir recursively
rg "pattern" path/              # Search specific path
rg -i "pattern"                 # Case-insensitive
rg -l "pattern"                 # List files only
rg -C 3 "pattern"               # Show 3 lines context
rg --type typescript "pattern"  # Search only .ts files
```

## fd
Fast file finder.
```bash
fd "name"                       # Find files/dirs matching name
fd -e ts                        # Find all .ts files
fd -H "pattern"                 # Include hidden files
fd -I "pattern"                 # Don't respect .gitignore
fd -x command                   # Execute command on results
```

## bat
Cat with syntax highlighting.
```bash
bat file.ts                     # View with highlighting
bat -l json                     # Force JSON highlighting
bat -n file.ts                  # Show line numbers
bat -A file.ts                  # Show all (including non-printable)
```

## fzf
Interactive fuzzy finder.
```bash
history | fzf                   # Search history
fd | fzf                        # Find and select file
git branch | fzf                # Select git branch
cat file | fzf                  # Filter file content
```

## eza
Modern ls replacement.
```bash
eza                             # Better ls
eza -la                         # Long format with hidden
eza -T                          # Tree view
eza -la --git                   # Show git status
eza --sort=modified             # Sort by modification time
```

## httpie (`http`)
User-friendly HTTP client.
```bash
http GET url                    # GET request
http POST url key=value         # POST JSON
http -f POST url key=value      # POST form data
http --headers url              # Show headers only
http --download url             # Download file
```

## tldr
Quick command examples (simpler than man pages).
```bash
tldr command                    # Get quick examples
tldr -u                         # Update cache
tldr -l                         # List all pages
```
